# Google Generative AI Story Generator

This project allows you to interact with Google Generative AI using Node.js to generate stories or other textual content based on user prompts. It includes both basic and streaming content generation modes.

## Features

- **Basic Content Generation**: Takes a user prompt and generates a complete story.
- **Streaming Content Generation**: Streams the story in chunks as it's being generated.

## Prerequisites

1. **Node.js**: Ensure you have Node.js installed on your system. You can download it [here](https://nodejs.org/).
2. **Google Generative AI API Key**: You need an API key from Google to use their Generative AI services.
3. **dotenv**: Used for loading environment variables from a `.env` file.
